Ini merupakan repositori proyek mata kuliah MKPL.
Software yang dibangun dalam proyek ini adalah sistem informasi perusahaan sewa peralatan multimedia.
Kebutuhan yang diminta oleh stakeholder adalah sistem dapat melakukan pengelolaan datainventaris perusahaan yang mana di dalamnya memuat fitur:
1. Lihat data status inventaris
2. Tambah data inventaris
3. Edit $ hapus data inventaris.
